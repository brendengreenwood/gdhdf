Plan

Disparate data sources⬡ Excel · Memory
Build
Open tool, build scenario
Enters the tool to translate strategy into a live scenario. Selects date, contract, commodity, inputs basis, max, freight.
Strategy to execution⬡ Pricing Tool
Model
Nudge inputs, shape updates
ML generates territory and bid prices as inputs change. No apply button needed.
Real-time modeling⬡ Pricing Tool · ML
Activate
Activate scenario
Cascades to lead lists, Salesforce bid prices, and producer accounts automatically.
Auto cascade⬡ Pricing Tool · Salesforce
Monitor
Monitor on dashboard
Contracts attributed to scenario automatically. Rejected bids accumulate in real time — merchant can see when a scenario is underperforming without waiting for a report.
Auto attribution⬡ Pricing Tool · ERP
Revise
Revise mid-week
Hit Revise, adjust, child scenario created. Closed leads excluded. All systems updated.
Mid-cycle control⬡ Pricing Tool
Close
Export performance, update position
Exports scenario performance data and manually incorporates closed contracts into position calculations for the following week. Tool boundary — position management still lives outside the system.
Manual reconciliation⬡ Pricing Tool · Excel
